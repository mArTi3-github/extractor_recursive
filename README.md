# extractor_recursive

Этот проект представляет собой программу на C# для рекурсивной распаковки архивов. Программа использует библиотеку SevenZipSharp для обработки различных форматов архивов.

## Особенности

- Рекурсивная распаковка архивов.
- Поддержка фильтрации по расширениям архивов.
- Отслеживание прогресса распаковки.
- Запись отладочной информации о распакованных и нераспакованных файлах.

## Установка

1. Убедитесь, что у вас установлен .NET SDK.
2. Клонируйте репозиторий или загрузите исходный код.
3. Установите библиотеку SevenZipSharp, если она не установлена.

## Использование

### Параметры командной строки

- `input_path`: Распаковка всех возможных файлов.
- `--extensions=ext1,ext2,... input_path`: Распаковка только файлов с указанными расширениями.


### Примеры

1. Распаковка всех файлов в директории:
   ```cmd
   extractor_recursive.exe "C:\path\to\archive"
   ```

2. Распаковка только .zip и .rar файлов:
   ```cmd
   extractor_recursive.exe --extensions=zip,rar "C:\path\to\archive"
   ```

## Вывод
После завершения работы программы будут созданы два файла:

- `_processed_archives.txt`: Список успешно распакованных архивов.
- `_failed_to_extract_files.txt`: Список файлов, которые не удалось распаковать.

## Требования
- .NET SDK
- SevenZipSharp
- 7z.dll (должен быть расположен в C:\Program Files\7-Zip\7z.dll)
